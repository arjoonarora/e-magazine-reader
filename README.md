# 📖 E-Magazine Reader

A fully responsive digital magazine reader with **flip mode**, **scroll mode**, **mobile optimization**, and **real book physics**.

🔗 **Live Demo:** https://arjoonarora.github.io/e-magazine-reader/

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **Flip Mode** | Two-page spread for desktop — like a real book |
| **Scroll Mode** | Vertical scroll — all pages in one column |
| **Mobile Single Page** | Optimized for phones — one page at a time |
| **Cover on Left** | Real book physics — cover on left, text on right |
| **Swipe Support** | Touch gestures for mobile navigation |
| **Keyboard Controls** | Arrow keys (← →) for desktop |
| **Fullscreen Mode** | Distraction-free reading experience |
| **22 Pages** | Supports page0 to page21 (22 images) |
| **Responsive** | Works on mobile, tablet, laptop, desktop |

---

## 🎮 How to Use

### Desktop (Flip Mode)
- Click **←** and **→** buttons to flip pages
- Or use **Arrow Left** and **Arrow Right** keys
- Two pages visible at once (spread)

### Mobile (Single Page)
- Swipe **left** → next page
- Swipe **right** → previous page
- One page visible at a time

### Scroll Mode
- Click **"📜 Scroll Mode"** button
- All pages stack vertically
- Scroll to read
- Click **"📖 Flip Mode"** to switch back

### Fullscreen
- Click **⛶** button for fullscreen mode
- Click **✕** to exit
